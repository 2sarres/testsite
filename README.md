# Mini CMS PHP (articles + admin + éditeur riche)

## Prérequis
- PHP 8.1+ (avec SQLite activé)

## Lancer en local (Windows)
Dans ce dossier :

```powershell
php -S 127.0.0.1:8000 -t public
```

Puis ouvrir `http://127.0.0.1:8000`.

## Installation
- Va sur `/install.php` pour créer la base et le premier compte admin.
- Après installation, supprime ou renomme `public/install.php` (recommandé).
